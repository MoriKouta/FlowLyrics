# FlowLyrics

[English](README.md) | [日本語](README.ja.md)

FlowLyricsは、Windows版Spotifyで再生中の曲に追従し、同期歌詞を常に手前へ表示するカスタマイズ可能な歌詞オーバーレイです。ロック中は歌詞部分のクリックを背後のアプリへ通せます。

> このパッケージは確認用の`1.3.0-dev.4`です。正式版`1.3.0`ではありません。

Spotify Developerアカウント、Client ID、Spotifyのパスワードは不要です。再生情報はWindowsのGlobal System Media Transport Controls（SMTC）から取得し、歌詞はLRCLIBで検索します。

## 主な機能

- Windows版Spotifyの曲変更、再生、一時停止、シークへ自動追従
- 曲名・アーティスト・版・曲尺を再検証する、安全性を優先したLRCLIB自動マッチング
- 日本語タイトルでローマ字歌詞を自動採用せず、日本語本文を優先する文字種判定
- 途中経過を順次表示する編集可能なLRCLIB候補検索、メタデータ付きプレビュー、曲ごとの手動選択保存
- 現在曲だけのキャッシュ削除と、「見つからない」キャッシュの1分制限
- LRCLIBの同期歌詞、ローカルキャッシュ、自動再試行
- 通常歌詞しかない場合の簡易タイミング表示（任意）
- ローカル`.lrc`とファイル変更の自動検出
- 背景透過／色付き背景、常に手前、ロック時のクリック透過
- 狭いウィンドウでも自動改行し、文字サイズを自動調整
- 表示行数1～12行、現在行の位置、文字揃え、行間、濃さを調整可能
- 10種類のカラープリセット、個別色指定、統一感のあるランダム配色、プレイヤーUIとSettingsで共通のアクセント色、全体／色別の軽量ブレンドモード
- フォント、縁取り、影、背景、枠、余白、角丸を調整可能
- 前の曲、再生／一時停止、次の曲、シーク、ミュート、音量操作
- 再生バーの左右に現在時間と曲尺を表示し、マウス位置の秒数もプレビュー
- 設定中もプレイヤー操作が使える非モーダルSettingsウィンドウ
- 四隅からのサイズ変更、タスクトレイ、Windows自動起動、ショートカットのON／OFF
- 10言語UI：英語、日本語、簡体字中国語、繁体字中国語、韓国語、スペイン語、フランス語、ドイツ語、ブラジルポルトガル語、ロシア語

## 対応環境

- Windows 10 バージョン1809以降、またはWindows 11
- Windows版Spotifyデスクトップアプリ
- 未取得の歌詞を検索するときのみインターネット接続

ポータブル版は自己完結型なので、別途.NETをインストールする必要はありません。

## インストールと起動

1. 提供された`FlowLyrics-v1.3.0-dev.4-win-x64-portable.zip`をダウンロードします。
2. ZIPを、書き込み可能なフォルダへ展開します。
3. `FlowLyrics.exe`を実行します。
4. Windows版Spotifyで曲を再生します。
5. 三点ボタンからSettingsを開き、言語や見た目を設定します。

現在の個人ビルドにはコード署名がないため、Windows SmartScreenが表示される場合があります。ダウンロード元を確認したうえで実行してください。

## 操作

| 操作 | 内容 |
| --- | --- |
| オーバーレイをドラッグ | ウィンドウを移動 |
| 四隅のグリップをドラッグ | ウィンドウサイズを変更 |
| 三点ボタンまたは右クリック | 設定や歌詞操作を表示 |
| プレイヤー操作部 | 前の曲、再生／一時停止、次の曲、シーク |
| VOLにマウスを重ねる | Spotify専用の縦型音量バーを表示 |
| VOLをクリック | Spotifyだけをミュート／解除 |
| `Ctrl + Alt + L` | ロック／ロック解除 |
| `Ctrl + Alt + K` | 表示／非表示 |
| トレイアイコンをダブルクリック | 表示／非表示 |

他のアプリと競合する場合、グローバルショートカットはSettingsで無効にできます。

ロック中は歌詞部分のマウス操作が背後のアプリへ通ります。表示中のプレイヤー操作部はロック中でも使用できます。

VOLは全アクティブ出力デバイスからSpotifyのWindows共有モード音声セッションだけを探し、システム全体の音量は変更しません。Spotifyが排他モード出力を使用中はWindowsがアプリ別音量を提供しないため、音量バーを無効化します。

## 歌詞の選択とローカルLRC

**Settings > 歌詞**では、現在のSpotify曲情報と、使用中のLRCLIBレコードを確認できます。歌詞やタイミングが違う場合や、安全な候補を1件に絞れなかった場合は、**LRCLIBから候補を選ぶ**を使用してください。Title、Artist、Album、Keywordは編集でき、英語名やローマ字名でも検索できます。

手動で選んだLRCLIB IDは同じSpotify曲に記憶されます。候補の**この歌詞を使用**を押すと、その曲の以前のキャッシュを削除して選択したLRCLIBレコードへ即座に差し替え、候補画面も閉じます。自動検索へ戻す場合だけ**手動選択を解除**を使用してください。

**Settings > 歌詞 > ローカルLRC**からLRCフォルダを開き、タイムスタンプ付き`.lrc`ファイルを入れてください。初期の保存場所は次のとおりです。

```text
%APPDATA%\FlowLyrics\lyrics-cache
```

ファイル名は次のどちらかにします。

```text
アーティスト - 曲名.lrc
曲名.lrc
```

起動中にLRCを追加、差し替え、削除しても自動検出します。同じフォルダに歌詞キャッシュの`.json`が存在しますが正常です。`.lrc`を一緒に入れて問題ありません。

## 歌詞が表示されない場合

1. Spotifyの現在の曲がWindowsのメディアパネルに表示されるか確認します。
2. Spotifyを再起動し、その後FlowLyricsを再起動します。
3. **Settings > 歌詞**を開き、候補がある場合はアーティスト、版、曲尺を確認して選びます。
4. 英語名／ローマ字名で再検索するか、タイムスタンプ付きLRCを追加します。

曖昧な結果は自動選択しません。曲尺差が2秒を超える、アーティストが明確に違う、Live／Instrumental／Remix／Cover／Karaoke／TV Sizeなどの版が違う場合は自動適用を禁止します。LRCLIBの混雑、アクセス制限、一時的なネットワーク障害では重複通信を止め、サーバーが指定した待機時間を守って自動再試行します。

## データとプライバシー

歌詞検索時に、曲名、アーティスト名、アルバム名、曲尺をLRCLIBへ送信します。音声、Spotifyの認証情報、パスワード、再生履歴一覧は送信しません。設定、ハッシュ名の歌詞キャッシュ、ログ、手動選択情報はPC内の次の場所に保存します。

```text
%APPDATA%\FlowLyrics
```

## 制限

- 同じWindows PC上のSpotifyメディアセッションへ追従します。スマートフォンや別PCだけで再生している曲には追従できません。
- 排他的フルスクリーンのアプリがオーバーレイより前に表示される場合があります。
- 曲検出と再生操作は、SpotifyがWindows SMTCへ公開している機能に依存します。
- Spotify Track IDはSMTCから取得できない環境があります。その場合は正規化した曲名・アーティスト・アルバム・曲尺から作る安定キーをキャッシュと手動選択に使用します。
- 同期歌詞がない曲には、通常歌詞の簡易表示またはローカルLRCが必要です。

## ソースからビルド

[.NET 10 SDK](https://dotnet.microsoft.com/download/dotnet/10.0)をインストールし、次を実行します。

```powershell
dotnet run --project FlowLyrics.csproj
```

マッチングとキャッシュの回帰テストは次で実行できます。

```powershell
dotnet run --project FlowLyrics.Tests/FlowLyrics.Tests.csproj -c Release
```

Windows x64向けの自己完結版は次のコマンドで作成できます。

```powershell
dotnet publish FlowLyrics.csproj -c Release -r win-x64 --self-contained true `
  -p:PublishSingleFile=true `
  -p:IncludeNativeLibrariesForSelfExtract=true `
  -p:PublishTrimmed=false
```
