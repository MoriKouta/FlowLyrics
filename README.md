# FlowLyrics

## ダウンロード

[FlowLyrics 最新版をダウンロード](https://github.com/MoriKouta/FlowLyrics/releases/latest)

Releaseページの「Assets」から、FlowLyricsのZIPファイルをダウンロードしてください。

※「Source code (zip)」「Source code (tar.gz)」はアプリ本体ではありません。

Windows版Spotifyで再生している曲に追従し、同期歌詞をデスクトップへ表示するWindows専用オーバーレイアプリです。

背景を完全に透明にして歌詞だけを表示したり、色付きパネルや縁取り、フォント、表示位置などを自由にカスタマイズできます。常に手前への表示やクリック透過にも対応しているため、作業中や映像編集、配信、デスクトップ鑑賞用として利用できます。

## 主な機能

* Windows版Spotifyの再生曲へ自動追従
* タイムスタンプ付き同期歌詞を表示
* 長い歌詞の自動改行と文字サイズ自動調整
* 背景透過、配色プリセット、ランダム配色
* フォント、文字色、背景、縁取り、影、透明度を調整
* 常に手前、クリック透過、ウィンドウロック
* 再生、一時停止、前後スキップ、シーク、音量操作
* 狭いウィンドウでも再生ボタンを維持
* 手持ちのLRCファイルを読み込み可能
* SpotifyのClient IDやDeveloperアカウント不要
* 取得済み歌詞をPC内へキャッシュ

## 対応環境

* Windows 10 1809以降／Windows 11
* Windows x64
* Windows版Spotifyデスクトップアプリ
* 新しい歌詞を取得するときのみインターネット接続が必要

## インストール

1. `FlowLyrics-v0.4.2-win-x64-portable.zip`をダウンロードします。
2. ZIPをすべて展開します。
3. `FlowLyrics.exe`を起動します。
4. Windows版Spotifyで曲を再生します。

自己完結型のため、.NET Runtimeの追加インストールは必要ありません。

初回起動時にWindows SmartScreenが表示される場合があります。本アプリは現在コード署名されていない個人制作アプリです。配布元を確認したうえで「詳細情報」から実行してください。

## 歌詞について

歌詞はLRCLIBから取得します。LRCLIBに同期歌詞が登録されていない曲や、曲名・バージョンを正しく判別できない曲では表示されない場合があります。その場合は、手持ちのLRCファイルを読み込んで使用できます。

検索時には曲名、アーティスト名、アルバム名、曲尺をLRCLIBへ送信します。Spotifyのアカウント情報、パスワード、音声データ、再生履歴一覧は取得・送信しません。
