# FlowLyrics v1.0.0

[English](#english) | [日本語](#日本語)

## English

Initial public release of FlowLyrics, a customizable synchronized-lyrics overlay for Spotify on Windows.

### Highlights

- Automatically follows the currently playing track in Spotify for Windows
- Displays timestamped synchronized lyrics from LRCLIB
- Transparent, always-on-top lyric overlay
- Adjustable font, colors, outline, shadow, opacity, line spacing, and panel appearance
- Multiple lyric lines with automatic wrapping and font-size adjustment
- Playback controls for previous track, play/pause, and next track
- Click-through lock mode for use over Nuke, browsers, streaming software, and other applications
- Per-track lyric timing adjustment
- Support for local `.lrc` files
- System tray support and global shortcuts
- No Spotify Developer account, Client ID, or password required

### Download

Download `FlowLyrics-v1.0.0-win-x64-portable.zip` from **Assets** below.

The automatically generated `Source code (zip)` and `Source code (tar.gz)` files are not the Windows application.

### Installation

1. Download the portable ZIP from Assets.
2. Extract the entire archive.
3. Run `FlowLyrics.exe`.
4. Start playing music in Spotify for Windows.

No additional .NET Runtime installation is required.

Windows SmartScreen may appear because this build is not code-signed. Verify that the file was downloaded from this repository before running it.

## 日本語

Windows版Spotify向け同期歌詞オーバーレイ「FlowLyrics」の初回公開版です。

### 主な機能

- Windows版Spotifyの再生曲へ自動追従
- LRCLIBのタイムスタンプ付き同期歌詞を表示
- 背景透過・常に手前の歌詞オーバーレイ
- フォント、配色、縁取り、影、透明度、行間、パネル表示を調整可能
- 複数行表示、長い歌詞の自動改行、文字サイズ自動調整
- 前の曲、再生・一時停止、次の曲をオーバーレイから操作
- Nuke、ブラウザ、配信ソフトなどの上で使えるクリック透過ロック
- 曲ごとの歌詞タイミング補正
- ローカル `.lrc` ファイルに対応
- タスクトレイ常駐、グローバルショートカット
- Spotify Developerアカウント、Client ID、パスワード不要

### ダウンロード

下の **Assets** から `FlowLyrics-v1.0.0-win-x64-portable.zip` をダウンロードしてください。

GitHubが自動表示する `Source code (zip)` と `Source code (tar.gz)` はアプリ本体ではありません。

### インストール

1. AssetsからポータブルZIPをダウンロードします。
2. ZIPをすべて展開します。
3. `FlowLyrics.exe` を起動します。
4. Windows版Spotifyで曲を再生します。

.NET Runtimeの追加インストールは不要です。

本ビルドはコード署名されていないため、Windows SmartScreenが表示される場合があります。このリポジトリからダウンロードしたファイルであることを確認してから実行してください。
